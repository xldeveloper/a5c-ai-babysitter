# avalonia-ui-setup

Set up Avalonia UI project for cross-platform desktop development.

## Quick Start

```javascript
const result = await invokeSkill('avalonia-ui-setup', {
  projectPath: '/path/to/project',
  projectName: 'MyAvaloniaApp',
  theme: 'fluent',
  mvvmFramework: 'reactiveui'
});
```

## Features

- True cross-platform (Windows, macOS, Linux)
- XAML UI similar to WPF
- ReactiveUI integration
- Fluent design theme
