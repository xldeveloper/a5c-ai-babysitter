# tauri-project-setup

Initialize Tauri project with Rust backend and frontend framework integration.

## Quick Start

```javascript
const result = await invokeSkill('tauri-project-setup', {
  projectPath: '/path/to/project',
  projectName: 'my-app',
  frontend: 'react',
  features: ['system-tray', 'auto-update', 'dialog']
});
```

## Features

- Rust backend
- Multiple frontend frameworks
- Cross-platform builds
- Plugin system
- Smaller bundle size than Electron
