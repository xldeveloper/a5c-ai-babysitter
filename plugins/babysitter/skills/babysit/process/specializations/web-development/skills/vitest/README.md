# Vitest Skill

Fast unit testing powered by Vite.

## Overview

Vitest is a blazing fast unit test framework powered by Vite.

## When to Use

- Unit testing
- Vite projects
- Jest migration
- Fast feedback loops

## Quick Start

```typescript
import { expect, test } from 'vitest';

test('adds 1 + 2 to equal 3', () => {
  expect(1 + 2).toBe(3);
});
```

## Integration

Works with react-testing-library-skill for React component testing.
