# Getopts Parser Generator Skill

Generate getopts-based argument parsing for shell scripts.

## Overview

This skill generates getopts-based argument parsing code for shell scripts, supporting both short and long options with validation.

## When to Use

- Adding argument parsing to shell scripts
- Supporting short and long options
- Validating argument values
- Generating help text

## Features

- Short option parsing
- Long option transformation
- Option value validation
- Required argument handling
- Help text generation
