# ShellCheck Config Generator Skill

Generate .shellcheckrc configuration for shell script linting.

## Overview

This skill generates ShellCheck configuration files with appropriate rules, exclusions, and documentation for consistent shell script linting.

## When to Use

- Setting up ShellCheck for a project
- Configuring linting rules
- Excluding specific warnings
- Documenting rule decisions

## Features

- .shellcheckrc generation
- Rule exclusion configuration
- Severity level settings
- Shell dialect selection
