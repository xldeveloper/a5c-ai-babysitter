# Cross-Platform Path Handler Skill

Generate cross-platform path handling utilities for CLI applications.

## Features

- Path separator normalization
- Home directory expansion
- Platform-specific config directories
- XDG Base Directory support

## Integration with Processes

| Process | Integration |
|---------|-------------|
| cross-platform-cli-compatibility | Path utilities |
| configuration-management-system | Config directories |
| cli-application-bootstrap | Path setup |
