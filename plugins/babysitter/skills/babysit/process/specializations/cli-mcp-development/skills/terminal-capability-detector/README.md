# Terminal Capability Detector Skill

Detect terminal capabilities for adaptive CLI output.

## Features

- Color level detection (none, basic, 256, truecolor)
- TTY status checking
- Terminal size detection
- Unicode support detection
- CI environment detection

## Integration with Processes

| Process | Integration |
|---------|-------------|
| cross-platform-cli-compatibility | Capability detection |
| cli-output-formatting | Adaptive output |
| progress-status-indicators | TTY awareness |
