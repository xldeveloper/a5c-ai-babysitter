# BATS Test Scaffolder Skill

Generate BATS test structure and fixtures for shell script testing.

## Overview

This skill scaffolds BATS testing infrastructure for shell scripts, including test files, fixtures, helpers, and custom assertions.

## When to Use

- Setting up BATS testing
- Creating test fixtures
- Implementing mock functions
- Generating test cases

## Features

- Test file generation
- Setup/teardown fixtures
- Custom assertions
- Mock helpers
- Test isolation
