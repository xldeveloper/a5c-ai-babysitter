# POSIX Shell Validator Skill

Validate scripts for POSIX compliance with bashism detection and portability checks.

## Overview

This skill validates shell scripts for POSIX compliance, detecting bashisms and suggesting portable alternatives for cross-platform compatibility.

## When to Use

- Validating scripts for POSIX compliance
- Detecting bashisms before deployment
- Ensuring cross-platform compatibility
- Generating compliance reports

## Features

- Bashism detection
- POSIX compliance checking
- Portable alternative suggestions
- Checkbashisms integration
