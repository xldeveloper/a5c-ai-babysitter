# FPGA Debugging Skill

On-chip debugging skill with ILA, VIO, and related FPGA debug tools.

## Purpose

This skill provides debugging expertise, enabling:
- Hardware-in-the-loop debugging
- Real-time signal observation
- Dynamic test pattern injection
- Systematic issue isolation

## Key Features

- ILA probe insertion and configuration
- Trigger condition setup
- VIO debug interface design
- Waveform capture and analysis
- ChipScope/SignalTap operation
- Debug/release build management

## Integration

Used primarily with:
- FPGA On-Chip Debugging Process
- Functional Simulation Process
- Design for Testability Process
