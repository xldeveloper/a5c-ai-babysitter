# Robotics and Simulation Engineering - References

## Academic Papers and Research

1. **Probabilistic Robotics**
   - https://ieeexplore.ieee.org/document/1389727
   - Foundational work on probabilistic approaches to robot localization and mapping

2. **The Dynamic Window Approach to Collision Avoidance**
   - https://ieeexplore.ieee.org/document/580977
   - Classic local navigation algorithm for mobile robots

3. **Rapidly-Exploring Random Trees: A New Tool for Path Planning**
   - http://msl.cs.uiuc.edu/~lavalle/papers/Lav98c.pdf
   - Foundational RRT algorithm for motion planning

4. **SLAM++: Simultaneous Localisation and Mapping at the Level of Objects**
   - https://www.cv-foundation.org/openaccess/content_cvpr_2013/papers/Salas-Moreno_SLAM_Simultaneous_Localisation_2013_CVPR_paper.pdf
   - Object-level SLAM approach

5. **ORB-SLAM: A Versatile and Accurate Monocular SLAM System**
   - https://ieeexplore.ieee.org/document/7219438
   - State-of-the-art visual SLAM system

6. **Domain Randomization for Transferring Deep Neural Networks from Simulation to the Real World**
   - https://arxiv.org/abs/1703.06907
   - Techniques for sim-to-real transfer

7. **Sim-to-Real: Learning Agile Locomotion For Quadruped Robots**
   - https://arxiv.org/abs/1804.10332
   - RL-based locomotion with sim-to-real transfer

8. **Deep Reinforcement Learning for Robotic Manipulation**
   - https://arxiv.org/abs/1610.00633
   - Deep RL applications in manipulation tasks

9. **Learning Dexterous In-Hand Manipulation**
   - https://arxiv.org/abs/1808.00177
   - OpenAI's work on dexterous manipulation

10. **A Survey of Multi-Agent Reinforcement Learning with Communication**
    - https://arxiv.org/abs/2203.08975
    - Multi-robot coordination and communication
