# Edge Deployment Skill

ML model optimization and deployment on robot edge devices (Jetson, embedded).

## Purpose

Provides expert capabilities for optimizing and deploying machine learning models on robot edge computing platforms with performance constraints.

## Key Features

- TensorRT optimization
- ONNX conversion
- INT8/FP16 quantization
- DeepStream integration
- DLA deployment
- Performance profiling

## When to Use

- Deploying models on Jetson
- Optimizing inference latency
- Quantizing models for edge
- Setting up video analytics
- Benchmarking edge performance
