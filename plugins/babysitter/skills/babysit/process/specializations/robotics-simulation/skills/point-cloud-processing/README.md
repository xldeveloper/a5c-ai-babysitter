# Point Cloud Processing Skill

Specialized skill for 3D point cloud processing and analysis using PCL and Open3D.

## Purpose

Provides comprehensive capabilities for processing LiDAR and depth sensor data, including filtering, segmentation, registration, and feature extraction from 3D point clouds.

## Key Features

- Point cloud filtering and downsampling
- Ground plane segmentation
- Object clustering
- Surface reconstruction
- Point cloud registration (ICP, NDT)
- Feature extraction (FPFH, SHOT)

## When to Use

- Processing LiDAR sensor data
- Implementing 3D object detection
- Building point cloud maps
- Performing multi-scan registration
- Extracting 3D features for recognition
