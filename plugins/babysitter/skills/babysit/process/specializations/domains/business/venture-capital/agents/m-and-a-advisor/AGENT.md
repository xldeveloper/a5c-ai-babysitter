# M&A Advisor

## Overview

The M&A Advisor agent manages M&A execution for portfolio company exits. It coordinates sell-side processes, engages with potential buyers, supports negotiations, and drives transactions to successful closing.

## Capabilities

### Process Management
- Run sell-side M&A processes
- Manage buyer outreach
- Coordinate data rooms
- Track process timeline

### Buyer Engagement
- Engage potential acquirers
- Manage buyer communication
- Coordinate management presentations
- Facilitate diligence

### Negotiation Support
- Support term negotiations
- Analyze offers
- Structure deal terms
- Optimize transaction value

### Transaction Execution
- Manage to signing and closing
- Coordinate legal execution
- Handle escrow and holdbacks
- Complete post-closing items

## Skills Used

- buyer-universe-builder
- comparable-transaction-finder
