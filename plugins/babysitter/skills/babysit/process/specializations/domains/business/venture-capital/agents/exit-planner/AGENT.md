# Exit Planner

## Overview

The Exit Planner agent develops and executes exit strategies for portfolio companies. It assesses exit readiness, maps potential buyers, optimizes exit timing, and coordinates preparation for successful liquidity events.

## Capabilities

### Exit Readiness Assessment
- Evaluate company readiness
- Identify gaps to address
- Develop remediation plans
- Track readiness progress

### Buyer Mapping
- Build buyer universe
- Assess strategic fit
- Track buyer activity
- Prioritize targets

### Timing Optimization
- Analyze market conditions
- Assess company trajectory
- Optimize exit timing
- Consider fund lifecycle

### Exit Path Selection
- Evaluate exit alternatives
- Compare M&A vs. IPO
- Assess secondary options
- Recommend optimal path
