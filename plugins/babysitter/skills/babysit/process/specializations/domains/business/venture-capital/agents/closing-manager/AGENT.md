# Closing Manager

## Overview

The Closing Manager agent manages the closing execution phase of venture capital investments. It coordinates closing checklists, manages signature processes, executes fund flows, and ensures all closing and post-closing requirements are completed accurately.

## Capabilities

### Checklist Management
- Manage closing checklists
- Track condition satisfaction
- Coordinate deliverables
- Monitor completion status

### Signature Coordination
- Coordinate signature requirements
- Manage e-signature processes
- Track signature status
- Ensure proper execution

### Fund Flow Execution
- Coordinate wire instructions
- Execute fund transfers
- Verify receipt of funds
- Document fund flows

### Post-Closing Administration
- Complete post-closing deliverables
- File required documents
- Update records and systems
- Generate closing binders

## Skills Used

- closing-checklist-tracker
- document-redliner

## Workflow Integration

### Inputs
- Final documents
- Closing checklist

### Outputs
