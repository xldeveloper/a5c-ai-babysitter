# IP Patent Analyzer

## Overview

The IP Patent Analyzer skill provides comprehensive intellectual property analysis for venture capital due diligence. It assesses patent portfolios, analyzes competitive IP landscapes, and evaluates freedom to operate to identify IP-related risks and opportunities.

## Capabilities

### Patent Portfolio Analysis
- Inventory and categorize patent assets
- Assess patent quality and enforceability
- Evaluate geographic coverage
- Analyze patent family relationships

### Competitive IP Landscape
- Map competitor patent holdings
- Identify potential infringement risks
- Track patent filing trends
- Assess white space opportunities

### Freedom to Operate (FTO)
- Identify relevant third-party patents
- Assess infringement risk
- Evaluate design-around options
- Recommend mitigation strategies

### IP Strategy Assessment
- Evaluate IP filing strategy
- Assess trade secret protection
- Review licensing agreements
- Analyze IP-related litigation risk

## Usage

### Analyze Patent Portfolio
```
Input: Patent numbers, company name, technology area
Process: Research patents, assess quality
Output: Portfolio analysis, strength assessment
```

### Map Competitive Landscape
```
Input: Technology domain, competitor list
Process: Search patent databases, map holdings
Output: Landscape map, competitive analysis
```

### Conduct FTO Analysis
```
Input: Product/technology description
Process: Search relevant patents, assess risk
Output: FTO report, risk assessment
```

### Review IP Strategy
```
Input: IP documentation, filing history
Process: Evaluate strategy, identify gaps
Output: Strategy assessment, recommendations
```

## Assessment Criteria

| Category | Key Factors |
|----------|-------------|
| Patent Quality | Claims breadth, prosecution history, citations |
| Portfolio Strength | Size, coverage, strategic value |
| FTO Risk | Infringement probability, enforcement likelihood |
| Trade Secrets | Protection measures, documentation |
| Licensing | Agreement terms, royalty obligations |

## Integration Points

- **Technical Due Diligence**: IP assessment within tech DD
- **Legal Due Diligence**: Coordinate with legal review
- **Tech Stack Scanner**: Connect technical and IP analysis
- **Legal Reviewer (Agent)**: Support legal agent

## Data Sources

- USPTO, EPO, WIPO patent databases
- Google Patents for search and analysis
- Paid services (Derwent, PatSnap, Cipher)
- Company IP documentation
- Litigation databases (PACER, Docket Navigator)

## Best Practices

1. Focus on patents relevant to core technology
2. Consider both defensive and offensive value
3. Assess IP in context of business model
4. Evaluate trade secrets alongside patents
5. Consider international IP strategy
