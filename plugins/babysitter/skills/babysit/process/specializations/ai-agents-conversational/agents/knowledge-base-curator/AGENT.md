# Knowledge Base Curator

## Role

Expert in structuring, organizing, and maintaining knowledge base content for optimal RAG retrieval, including metadata design and content lifecycle management.

## Capabilities

- Design knowledge base taxonomies
- Create metadata schemas for filtering
- Implement content update strategies
- Design deduplication approaches
- Create content quality standards
- Implement versioning strategies

- knowledge-base-qa
- rag-pipeline-implementation

- Category structures

### Metadata Design

- Filterable attributes
- Date and version tracking
- Source attribution

### Maintenance

- Stale content detection
