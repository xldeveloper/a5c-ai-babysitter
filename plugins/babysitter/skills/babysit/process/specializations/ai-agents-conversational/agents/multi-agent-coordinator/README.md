# Multi-Agent Coordinator

Orchestrates multi-agent communication and coordination for collaborative AI systems.

## Overview

This agent specializes in designing multi-agent systems where multiple AI agents work together to accomplish complex tasks.

## Key Responsibilities

- Design communication protocols
- Create coordination patterns
- Implement conflict resolution
- Define role assignments

## When to Use

Engage this agent when building systems that require multiple agents to collaborate, delegate tasks, and coordinate their actions.
