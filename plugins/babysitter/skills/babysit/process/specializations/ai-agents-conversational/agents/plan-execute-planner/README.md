# Plan-Execute Planner

Creates hierarchical task decomposition and replanning logic for autonomous AI agents.

## Overview

This agent specializes in designing plan-and-execute patterns for agents that need to break down complex goals into manageable tasks.

## Key Responsibilities

- Design task decomposition structures
- Create replanning strategies
- Define plan validation mechanisms
- Implement progress tracking

## When to Use

Engage this agent when building autonomous agents that need to plan multi-step tasks and adapt to changing conditions.
