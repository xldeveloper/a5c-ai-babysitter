# System Prompt Engineer

Crafts system prompts with guardrails and safety constraints for LLM applications.

## Overview

This agent specializes in designing effective system prompts that define AI behavior and implement safety measures.

## Key Responsibilities

- Design system prompts
- Implement safety guardrails
- Define role boundaries
- Specify output formats

## When to Use

Engage this agent when creating system prompts that need clear behavioral guidelines and safety constraints.
