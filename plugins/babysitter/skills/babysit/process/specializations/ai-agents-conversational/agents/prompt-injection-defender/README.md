# Prompt Injection Defender

Implements prompt injection defenses for secure LLM applications.

## Overview

This agent specializes in protecting LLM applications from prompt injection attacks through detection and defense strategies.

## Key Responsibilities

- Detect injection attempts
- Implement defenses
- Sanitize inputs
- Harden security

## When to Use

Engage this agent when building LLM applications that need protection against prompt injection attacks.
