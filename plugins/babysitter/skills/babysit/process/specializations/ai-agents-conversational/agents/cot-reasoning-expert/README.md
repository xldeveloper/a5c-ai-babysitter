# Chain-of-Thought Reasoning Expert

Designs chain-of-thought and reasoning prompts for complex problem solving.

## Overview

This agent specializes in creating prompts that elicit step-by-step reasoning for improved problem solving.

## Key Responsibilities

- Design CoT prompts
- Create reasoning templates
- Implement self-consistency
- Verify reasoning quality

## When to Use

Engage this agent when building systems that require complex reasoning or step-by-step problem decomposition.
