# LangChain Memory

Integrates LangChain memory components for conversation state management and context retention.

## Overview

This skill provides comprehensive memory integration for LangChain-based applications, supporting various memory types from simple buffers to sophisticated vector-based semantic retrieval systems.

## Key Features

- Multiple memory type implementations
- Configurable retention strategies
- Persistence and serialization support
- Integration with vector stores for semantic memory
- Token-aware memory management

## Usage

Use this skill when building conversational systems that need to maintain context across interactions.
