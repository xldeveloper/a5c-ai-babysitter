# Weaviate Integration

Integrates Weaviate vector database with GraphQL queries and hybrid search.

## Overview

This skill provides Weaviate vector database integration, enabling schema management, GraphQL queries, and hybrid search capabilities for RAG applications.

## Key Features

- Schema definition and management
- GraphQL query interface
- Hybrid search (vector + BM25)
- Multi-tenancy support
- Batch import operations

## Usage

Use this skill when building RAG applications that require Weaviate's hybrid search and GraphQL capabilities.
