# LangFuse Integration

Integrates LangFuse for LLM observability, tracing, and cost tracking.

## Overview

This skill provides LangFuse integration, enabling comprehensive LLM observability with tracing, prompt management, and analytics.

## Key Features

- LLM call tracing
- Cost tracking
- Prompt versioning
- Evaluation datasets
- Quality scoring

## Usage

Use this skill when building LLM applications that require observability and cost management.
