# Chroma Integration

Integrates Chroma local vector database for development and production RAG applications.

## Overview

This skill provides Chroma vector database integration, enabling local vector storage with multiple deployment modes suitable for development through production.

## Key Features

- Multiple deployment modes
- Collection management
- Metadata filtering
- Multi-tenancy support
- LangChain integration

## Usage

Use this skill when building RAG applications that need a flexible, easy-to-deploy vector database.
