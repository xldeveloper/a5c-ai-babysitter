# OpenTelemetry LLM

Configures OpenTelemetry instrumentation for LLM applications with distributed tracing.

## Overview

This skill provides OpenTelemetry configuration for LLM applications, enabling distributed tracing with LLM-specific semantic conventions.

## Key Features

- OpenTelemetry SDK setup
- LLM semantic conventions
- Multiple exporters
- Auto-instrumentation
- Context propagation

## Usage

Use this skill when building LLM applications that require integration with existing observability infrastructure.
