# LangChain Retriever

Implements LangChain retriever patterns for document retrieval in RAG applications.

## Overview

This skill provides retriever implementation capabilities for RAG (Retrieval-Augmented Generation) pipelines, supporting various retrieval strategies from basic vector search to advanced multi-query and ensemble approaches.

## Key Features

- Multiple retriever strategy implementations
- Configurable search parameters
- Metadata filtering support
- Result compression and reranking
- Parent document retrieval patterns

## Usage

Use this skill when building RAG pipelines that need flexible and powerful document retrieval capabilities.
