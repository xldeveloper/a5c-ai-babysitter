# Milvus Integration

Integrates Milvus distributed vector database for large-scale RAG applications.

## Overview

This skill provides Milvus vector database integration, enabling distributed vector storage with various index types for large-scale RAG applications.

## Key Features

- Multiple deployment modes
- Various index type support
- Partition management
- GPU acceleration
- Large-scale data handling

## Usage

Use this skill when building large-scale RAG applications that require distributed vector storage with high throughput.
