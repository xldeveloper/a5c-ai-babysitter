# Prompt Injection Detector

Detects and prevents prompt injection attacks in LLM applications.

## Overview

This skill provides prompt injection detection, enabling secure LLM applications through multiple detection and prevention strategies.

## Key Features

- Multiple detection methods
- Pattern matching
- ML-based detection
- Canary tokens
- Defense layering

## Usage

Use this skill when building secure LLM applications that need protection against prompt injection attacks.
