# Pinecone Integration

Integrates Pinecone vector database for RAG applications.

## Overview

This skill provides Pinecone vector database integration, enabling index management, vector operations, and metadata filtering for production RAG applications.

## Key Features

- Index creation and management
- Batch upsert operations
- Metadata filtering queries
- Namespace multi-tenancy
- Performance optimization

## Usage

Use this skill when building RAG applications that require Pinecone as the vector storage backend.
