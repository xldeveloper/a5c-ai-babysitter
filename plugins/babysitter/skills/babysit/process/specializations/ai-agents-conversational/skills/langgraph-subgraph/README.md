# LangGraph Subgraph

Enables subgraph composition and modular workflow design for LangGraph applications.

## Overview

This skill provides subgraph composition capabilities for LangGraph, enabling modular workflow design through reusable graph components that can be composed into larger systems.

## Key Features

- Modular subgraph design
- State mapping between graphs
- Reusability patterns
- Error isolation boundaries
- Parallel subgraph execution

## Usage

Use this skill when building complex LangGraph workflows that benefit from modular, reusable components.
