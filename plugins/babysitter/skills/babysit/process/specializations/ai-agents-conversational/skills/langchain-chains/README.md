# LangChain Chains

Composes LangChain chains using LCEL and legacy chain patterns for complex workflows.

## Overview

This skill provides chain composition capabilities using LangChain Expression Language (LCEL) and traditional chain patterns, enabling the creation of complex multi-step workflows for conversational AI applications.

## Key Features

- LCEL pipeline composition
- Sequential and parallel execution
- Conditional routing logic
- Fallback and retry mechanisms
- Streaming support

## Usage

Use this skill when building multi-step conversational workflows or complex dialogue systems that require chain composition.
