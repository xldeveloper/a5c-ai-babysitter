# RAG Embedding Generation

Generates embeddings with batch processing, caching, and multi-provider support.

## Overview

This skill provides embedding generation capabilities for RAG pipelines, supporting multiple embedding providers with batch processing, caching, and rate limiting for production-scale operations.

## Key Features

- Multi-provider embedding support
- Batch processing optimization
- Embedding caching
- Rate limiting and retries
- Quality validation

## Usage

Use this skill when building RAG pipelines that require efficient, scalable embedding generation.
