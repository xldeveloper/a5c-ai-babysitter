# Phoenix Arize Setup

Sets up Arize Phoenix observability platform for LLM debugging and evaluation.

## Overview

This skill provides Phoenix setup capabilities, enabling LLM observability with tracing, embedding analysis, and LLM-as-judge evaluations.

## Key Features

- OpenTelemetry tracing
- LLM-as-judge evals
- Embedding visualization
- Retrieval analysis
- Experiment management

## Usage

Use this skill when building LLM applications that require debugging tools and evaluation frameworks.
