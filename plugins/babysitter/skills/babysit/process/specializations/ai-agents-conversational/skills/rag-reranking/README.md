# RAG Reranking

Implements cross-encoder reranking and diversity filtering for improved retrieval quality.

This skill provides reranking capabilities for RAG pipelines, enabling improved retrieval quality through cross-encoder models, MMR diversity filtering, and multi-stage retrieval patterns.

- Cross-encoder reranking models
- MMR diversity filtering

- Score normalization
- Multi-stage retrieval pipelines

## Usage

Use this skill when optimizing RAG retrieval quality through result reranking and diversity filtering.
