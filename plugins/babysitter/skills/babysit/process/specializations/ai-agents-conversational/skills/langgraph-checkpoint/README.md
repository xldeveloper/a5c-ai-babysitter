# LangGraph Checkpoint

Configures LangGraph checkpointing and persistence for stateful workflow management.

## Overview

This skill provides checkpoint configuration capabilities for LangGraph workflows, enabling state persistence, recovery, and replay functionality across various storage backends.

## Key Features

- Multiple checkpoint backend support
- State serialization and compression
- Recovery and replay mechanisms
- Checkpoint versioning
- Retention policy management

## Usage

Use this skill when building stateful LangGraph workflows that require persistence and recovery capabilities.
