# Scope Permission Designer

Design and implement scoped permission models.

## Purpose

This skill designs OAuth scopes and permission models that enable fine-grained, business-aligned access control for APIs.

## Key Features

- Scope hierarchy design
- Permission validation
- Scope documentation
- Consent flow support
- Resource-level permissions

## When to Use

- Designing API authorization
- Implementing OAuth scopes
- Building consent flows
- Documenting permissions
