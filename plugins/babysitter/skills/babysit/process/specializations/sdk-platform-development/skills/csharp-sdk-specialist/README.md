# C#/.NET SDK Specialist

C#/.NET SDK development with async patterns.

## Purpose

This skill develops production-ready .NET SDKs with proper async/await patterns, multi-targeting support, and NuGet distribution.

## Key Features

- Async/await with cancellation
- Multi-targeting (.NET Standard, Core, 6+)
- NuGet packaging
- Dependency injection support
- Source link debugging

## When to Use

- Building .NET client libraries
- Implementing async API clients
- Creating NuGet packages
- Supporting multiple .NET versions
