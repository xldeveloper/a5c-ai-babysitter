# API Key Manager

API key generation, rotation, and management system.

## Purpose

This skill implements secure API key management with generation, rotation, usage tracking, and quota enforcement capabilities.

## Key Features

- Secure key generation
- Key rotation policies
- Usage tracking
- Quota enforcement
- Scoped permissions

## When to Use

- Implementing API authentication
- Building key management UIs
- Setting up usage quotas
- Managing test/live keys
