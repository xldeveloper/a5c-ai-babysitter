# Codemod Generator

Generate automated code migration scripts (codemods).

## Purpose

This skill creates automated code transformation scripts that help developers migrate their code to new SDK versions with minimal manual effort.

## Key Features

- AST-based transformations
- Multi-language support
- Dry-run previews
- Migration reports
- Rollback support

## When to Use

- Releasing major SDK versions
- Renaming APIs or methods
- Restructuring SDK interfaces
- Helping consumers upgrade
