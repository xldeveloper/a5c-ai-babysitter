# OAuth Flow Implementer

Implement OAuth 2.0 and OpenID Connect flows for SDKs.

## Purpose

This skill implements secure OAuth authentication flows for SDKs, handling authorization, token management, and refresh transparently.

## Key Features

- Authorization code with PKCE
- Client credentials flow
- Device authorization flow
- Automatic token refresh
- Secure token storage

## When to Use

- Adding OAuth to SDKs
- Implementing PKCE for mobile/SPA
- Supporting multiple auth flows
- Building CLI authentication
