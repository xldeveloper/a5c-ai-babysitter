# Deprecation Manager

Manage API and SDK deprecation lifecycle.

## Purpose

This skill manages the complete deprecation lifecycle, ensuring API consumers have adequate notice and migration support before features are sunset.

## Key Features

- Deprecation timeline tracking
- SDK deprecation warnings
- Consumer notifications
- Migration documentation
- Usage monitoring

## When to Use

- Deprecating API endpoints
- Sunsetting SDK features
- Managing version transitions
- Communicating breaking changes
