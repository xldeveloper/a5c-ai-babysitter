# GraphQL Schema Designer

GraphQL schema design and optimization with federation support.

## Purpose

This skill helps design robust, scalable GraphQL schemas that follow industry best practices and support modern patterns like Apollo Federation.

## Key Features

- Type-safe schema design
- Apollo Federation support
- Query complexity optimization
- Automatic documentation generation
- Relay-compatible pagination

## When to Use

- Designing new GraphQL APIs
- Migrating REST APIs to GraphQL
- Implementing federated microservices
- Optimizing existing GraphQL schemas
