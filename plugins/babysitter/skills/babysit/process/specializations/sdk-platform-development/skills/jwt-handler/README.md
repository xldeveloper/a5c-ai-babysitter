# JWT Handler

JWT creation, validation, and management for SDK authentication.

## Purpose

This skill implements secure JWT-based authentication with proper key rotation, claims validation, and token management.

## Key Features

- Multi-algorithm support
- JWKS key rotation
- Claims validation
- Token refresh handling
- Encryption support (JWE)

## When to Use

- Implementing JWT authentication
- Setting up JWKS endpoints
- Managing token lifecycle
- Validating API tokens
