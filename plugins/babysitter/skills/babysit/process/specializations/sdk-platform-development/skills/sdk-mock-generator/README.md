# SDK Mock Generator

Generate mock servers and clients for SDK testing.

## Purpose

This skill creates mock servers and test fixtures that enable thorough SDK testing without depending on production APIs.

## Key Features

- OpenAPI-based mock generation
- Stateful mocking support
- Error simulation
- Request recording
- Latency simulation

## When to Use

- Testing SDK integrations
- Building sandbox environments
- Creating test fixtures
- Simulating error conditions
