# Java SDK Specialist

Java SDK development for enterprise environments.

## Purpose

This skill develops enterprise-grade Java SDKs with builder patterns, reactive support, and Maven Central distribution.

## Key Features

- Builder pattern for configuration
- Reactive and sync clients
- Maven/Gradle publishing
- Java 11+ compatibility
- Proper exception handling

## When to Use

- Building enterprise Java clients
- Implementing reactive API clients
- Creating Maven Central packages
- Supporting legacy Java environments
