# Security Review Agent

Reviews SDK authentication and security implementations.

## Purpose

This agent ensures SDK authentication and security implementations follow best practices and are free from vulnerabilities.

## Key Responsibilities

- Auditing authentication flows
- Reviewing credential handling
- Identifying vulnerabilities
- Recommending security improvements

## When to Engage

- Reviewing authentication implementations
- Auditing SDK security
- Assessing credential handling
- Validating security fixes
