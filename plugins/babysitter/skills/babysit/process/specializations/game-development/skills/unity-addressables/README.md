# Unity Addressables Skill

## Overview

The Unity Addressables skill provides asset management capabilities for efficient content loading, remote delivery, and memory optimization.

## Purpose

Addressables enables scalable asset management. This skill provides:

- **Async Loading**: Non-blocking asset loading
- **Remote Content**: CDN-hosted asset bundles
- **Content Updates**: Patch games without full rebuilds
- **Memory Control**: Fine-grained asset lifecycle management

## Use Cases

1. **DLC Systems**: Downloadable content delivery
2. **Memory Optimization**: Load/unload assets dynamically
3. **Live Updates**: Update content without app store reviews
4. **Large Projects**: Manage thousands of assets

- **Performance Optimization** (`performance-optimization.js`)
