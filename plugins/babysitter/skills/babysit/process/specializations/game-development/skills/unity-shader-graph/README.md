# Unity Shader Graph Skill

## Overview

The Unity Shader Graph skill enables visual shader authoring for creating custom materials and effects without writing code.

## Purpose

Shader Graph democratizes shader development. This skill provides:

- **Visual Authoring**: Node-based shader creation
- **Real-Time Preview**: See changes instantly
- **Pipeline Integration**: URP and HDRP support
- **Extensibility**: Custom nodes and subgraphs

## Use Cases

1. **Stylized Materials**: Toon, cel-shading
2. **Environmental Effects**: Water, foliage
3. **Special Effects**: Dissolve, hologram
4. **Procedural Textures**: Noise-based patterns

## Processes That Use This Skill

- **VFX Implementation** (`vfx-implementation.js`)
- **Art Asset Pipeline** (`art-asset-pipeline.js`)

## Related Skills

- `unity-urp`: URP pipeline integration
- `unity-hdrp`: HDRP pipeline integration
- `shader-effects`: Effect implementation
