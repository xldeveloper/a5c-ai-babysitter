# Unity URP Skill

## Overview

The Unity URP skill provides comprehensive integration with Unity's Universal Render Pipeline for configuring rendering settings, creating custom shaders, and implementing post-processing effects.

## Purpose

URP is Unity's scriptable render pipeline optimized for performance across platforms. This skill enables:

- **Pipeline Configuration**: Set up URP assets and renderers
- **Shader Development**: Create URP-compatible shaders
- **Post-Processing**: Configure visual effects and color grading
- **Custom Features**: Implement render features and passes

## Use Cases

1. **Visual Style Setup**: Configure rendering for desired art style
2. **Performance Optimization**: Tune pipeline for target platforms
3. **Effect Implementation**: Create custom visual effects
4. **Quality Tiers**: Set up scalable quality settings
