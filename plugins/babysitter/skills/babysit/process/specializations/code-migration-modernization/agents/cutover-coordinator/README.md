# Cutover Coordinator Agent

## Overview

The Cutover Coordinator agent manages production cutover activities. It handles planning, stakeholder coordination, and go/no-go decision making.

## Quick Start

1. **Plan cutover sequence**
2. **Coordinate stakeholders**
3. **Execute cutover**
4. **Validate success**

## Features

- Cutover planning and sequencing
- Stakeholder communication
- Go/no-go decision criteria
- Rollback coordination
