# Technical Debt Auditor Agent

## Overview

The Technical Debt Auditor agent provides comprehensive technical debt assessment. It quantifies debt impact, estimates remediation costs, and recommends prioritized action plans.

## Quick Start

### Prerequisites

- Codebase access
- Quality analysis tools
- Historical data (optional)

### Basic Usage

1. **Run debt audit**
2. **Review inventory**
3. **Prioritize remediation**
4. **Plan sprints**

## Features

### Debt Categories

| Category | Examples | Impact |
|----------|----------|--------|
| Code | Complexity, duplication | Productivity |
| Architecture | Coupling, layers | Flexibility |
| Test | Coverage gaps | Quality |
| Documentation | Missing docs | Onboarding |

### Outputs

- Debt inventory
- Prioritized backlog
- ROI analysis
- Remediation roadmap
