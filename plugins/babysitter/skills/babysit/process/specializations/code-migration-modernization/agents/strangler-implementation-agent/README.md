# Strangler Implementation Agent

## Overview

The Strangler Implementation Agent executes the strangler fig pattern for gradual migration. It manages traffic routing, parallel running, and coordinated cutover.

## Quick Start

1. **Set up routing layer**
2. **Extract features incrementally**
3. **Manage traffic transition**
4. **Complete cutover**

## Features

- Traffic splitting and routing
- Parallel system running
- Incremental cutover
- Legacy decommissioning
