# Cloud Migration Engineer Agent

## Overview

The Cloud Migration Engineer agent executes cloud migration initiatives. It handles infrastructure provisioning, containerization, and operational setup.

## Quick Start

### Prerequisites

- Cloud account access
- Source application access
- Target architecture defined

### Basic Usage

1. **Provision infrastructure**
2. **Containerize apps**
3. **Migrate data**
4. **Configure operations**

## Features

### Migration Types

| Type | Description | Effort |
|------|-------------|--------|
| Rehost | Lift and shift | Low |
| Replatform | Optimize for cloud | Medium |
| Refactor | Cloud-native | High |

### Capabilities

- IaC generation
- Container creation
- Data migration
- Security setup
