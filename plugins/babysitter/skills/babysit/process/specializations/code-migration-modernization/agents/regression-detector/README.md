# Regression Detector Agent

## Overview

The Regression Detector agent identifies regressions introduced during migration. It compares behavior, performance, and visual output to ensure migration quality.

## Quick Start

1. **Establish baseline**
2. **Run comparison tests**
3. **Analyze regressions**
4. **Report findings**

## Features

- Functional regression detection
- Performance regression monitoring
- Visual regression testing
- API contract verification
