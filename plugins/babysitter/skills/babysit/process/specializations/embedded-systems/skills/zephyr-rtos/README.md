# Zephyr RTOS Skill

Specialized skill for Zephyr RTOS development and configuration.

## Purpose

Provides comprehensive Zephyr RTOS development support including:
- Device tree overlay generation
- Kconfig configuration management
- West build system operation
- Networking and Bluetooth stack setup
- Power management configuration
- MCUboot secure boot integration

## Key Features

- Device tree and binding management
- Kconfig option configuration
- Multi-image build support
- Bluetooth LE and Mesh configuration
- TCP/IP networking setup
- Logging and shell configuration

## Integration

Used by processes requiring Zephyr expertise:
- RTOS integration projects
- Secure boot implementation
- OTA firmware updates
- Low-power device design

## Dependencies

- Zephyr SDK
- West meta-tool
- Device tree compiler
- CMake and Ninja build system
