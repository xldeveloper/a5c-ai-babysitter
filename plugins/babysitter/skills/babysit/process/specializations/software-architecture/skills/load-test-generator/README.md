# Load Test Generator

Generates load test scripts for k6, Locust, and Gatling from OpenAPI specs.

## Features

- Multi-framework support (k6, Locust, Gatling, Artillery)
- OpenAPI spec to test generation
- Ramp-up/ramp-down patterns
- Threshold configuration
- Test data handling

## Usage

Define scenarios or provide OpenAPI spec. Generates ready-to-run load test scripts.
