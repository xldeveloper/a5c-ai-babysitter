# Performance Engineer Agent

## Overview

Specialized agent for performance engineering including profiling, bottleneck identification, optimization strategies, and load testing interpretation.

## Capabilities

- Conduct performance profiling
- Identify bottlenecks from metrics
- Recommend optimization strategies
- Design load testing scenarios
- Interpret APM tool data
- Plan capacity requirements
- Model performance characteristics

## Target Processes

- performance-optimization
