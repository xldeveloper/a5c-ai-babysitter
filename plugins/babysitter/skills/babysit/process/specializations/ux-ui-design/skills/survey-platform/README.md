# Survey Platform Skill

Integrate with survey platforms for user research data collection and analysis.

## Overview

This skill enables integration with popular survey platforms to create, distribute, and analyze user surveys with standardized usability metrics.

## Key Features

- **Multi-Platform Support**: Typeform, SurveyMonkey, Google Forms
- **Standard Metrics**: SUS, NPS, CSAT calculations
- **Survey Creation**: Generate surveys programmatically
- **Response Analysis**: Statistical analysis and insights
- **Report Generation**: Automated reporting

## When to Use

- Conducting post-task usability surveys
- Measuring user satisfaction over time
- Collecting quantitative user research data
- Benchmarking against industry standards

## Standardized Metrics

| Metric | Range | Interpretation |
|--------|-------|----------------|
| SUS | 0-100 | >68 = Above average |
| NPS | -100 to 100 | >0 = More promoters |
| CSAT | 0-100% | >80% = Good |
