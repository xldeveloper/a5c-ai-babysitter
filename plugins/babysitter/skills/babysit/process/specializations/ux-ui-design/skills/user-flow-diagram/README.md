# User Flow Diagram Skill

Generate user flow diagrams and navigation flowcharts for UX documentation.

## Overview

This skill creates visual representations of user flows, navigation structures, and application state machines using Mermaid.js and other diagramming formats.

## Key Features

- **Multiple Formats**: Flowcharts, sequence diagrams, state machines
- **Mermaid.js Support**: Generate markdown-compatible diagrams
- **Export Options**: SVG, PNG, PDF output
- **Flow Validation**: Detect dead ends and unreachable nodes
- **Statistics**: Flow complexity metrics

## When to Use

- Documenting user journeys
- Creating navigation documentation
- Visualizing application state flows
- Information architecture planning
- Validating user path completeness

## Diagram Types

| Type | Use Case | Example |
|------|----------|---------|
| Flowchart | Navigation flows | Page-to-page navigation |
| Sequence | Interactions | API call flows |
| State | UI states | Form validation states |
| Journey | User experience | End-to-end flows |
