# Accessibility Report Skill

Generate VPAT, ACR, and other accessibility compliance documentation.

## Overview

This skill automates the creation of accessibility compliance reports required for enterprise and government procurement processes.

## Key Features

- **VPAT Generation**: Create VPAT 2.4 documents
- **ACR Creation**: Accessibility Conformance Reports
- **WCAG Checklists**: Complete success criteria tracking
- **Remediation Roadmaps**: Prioritized fix recommendations
- **Multi-Format Export**: DOCX, PDF, HTML, Markdown

## When to Use

- Preparing for enterprise sales/procurement
- Government contract compliance
- Annual accessibility audits
- Tracking remediation progress

## Report Types

| Type | Purpose | Audience |
|------|---------|----------|
| VPAT | Product accessibility declaration | Procurement |
| ACR | Detailed conformance report | Technical teams |
| WCAG Checklist | Success criteria tracking | Dev teams |
| Roadmap | Remediation planning | Management |

## Conformance Levels

- **Supports**: Fully meets the criterion
- **Partially Supports**: Some functionality meets
- **Does Not Support**: Does not meet
