# Color Palette Generator Skill

Generate accessible color palettes with WCAG compliance for design systems and component libraries.

## Overview

This skill automates the creation of accessible color palettes, ensuring all color combinations meet WCAG contrast requirements while maintaining visual harmony.

## Key Features

- **Color Scale Generation**: Create 50-900 color scales from base colors
- **WCAG Compliance**: Automatic contrast ratio validation
- **Scheme Generation**: Complementary, analogous, triadic, and monochromatic schemes
- **Dark Mode Support**: Generate dark mode variants automatically
- **Token Export**: Export to CSS, SCSS, JSON, or W3C design token format

## When to Use

- Starting a new design system
- Auditing existing color palettes for accessibility
- Creating dark mode color variants
- Generating color tokens for multi-platform projects
