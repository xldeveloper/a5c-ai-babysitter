# A/B Testing Agent

Design and analyze A/B tests for UX optimization.

## Overview

This agent designs statistically rigorous A/B tests and analyzes results to support data-driven UX decisions.

## Key Capabilities

- **Hypothesis Design**: Clear experiment formulation
- **Sample Size Calculation**: Statistical power planning
- **Significance Testing**: P-value and confidence intervals
- **Result Interpretation**: Actionable insights

## When to Use

- Planning UX experiments
- Analyzing test results
- Validating design changes
- Making data-driven decisions
