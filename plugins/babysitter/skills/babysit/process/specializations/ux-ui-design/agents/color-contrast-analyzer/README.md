# Color Contrast Analyzer Agent

Analyze and optimize color contrast for accessibility compliance.

## Overview

This agent specializes in color accessibility analysis, helping ensure that color choices meet WCAG requirements and work for users with color vision deficiencies.

## Key Capabilities

- **Contrast Calculation**: WCAG 2.1 ratio computation
- **CVD Simulation**: Color blindness visualization
- **Alternative Suggestions**: Passing color recommendations
- **Palette Analysis**: Full color system audits

## When to Use

- Validating design system colors
- Fixing contrast failures
- Creating accessible color palettes
- Dark mode color verification
