# Design Documentation Agent

Generate comprehensive design documentation for components and systems.

## Overview

This agent creates maintainable design documentation that enables consistent implementation and adoption across teams.

## Key Capabilities

- **Component Specs**: Detailed specifications
- **Usage Guidelines**: Dos and don'ts
- **Code Examples**: Framework-specific snippets
- **Accessibility Notes**: A11y requirements

## When to Use

- Building design system documentation
- Creating component specifications
- Documenting design patterns
- Maintaining design changelogs
