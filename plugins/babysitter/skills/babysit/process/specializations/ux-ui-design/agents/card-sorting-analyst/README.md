# Card Sorting Analyst Agent

Conduct and analyze card sorting studies for information architecture.

## Overview

This agent analyzes card sorting study results to inform information architecture decisions based on user mental models.

## Key Capabilities

- **Open/Closed Sorts**: Support both methodologies
- **Similarity Analysis**: Card relationship matrices
- **Dendrogram Generation**: Hierarchical clustering
- **IA Recommendations**: Actionable structure suggestions

## When to Use

- Planning site navigation
- Validating content categories
- Understanding user mental models
- Improving findability
