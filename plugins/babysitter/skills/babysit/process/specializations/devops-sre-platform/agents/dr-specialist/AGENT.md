## DR Planning Framework

When planning DR, this agent considers:
1. **Identify**: Critical systems and dependencies
2. **Protect**: Backup and replication strategies
3. **Detect**: Monitoring and alerting for disasters
4. **Respond**: Failover procedures
5. **Recover**: Full service restoration
6. **Learn**: Post-incident improvements
