# Data Structure Selector

A skill for selecting optimal data structures.

## Overview

This skill analyzes the required operations and constraints to recommend the most efficient data structure for the given use case.

## Key Features

- Operation-based selection
- Complexity analysis
- Trade-off evaluation
- Augmentation suggestions
- Alternative comparisons

## Common Data Structures

- Arrays and Vectors
- Hash Maps/Sets
- Binary Search Trees
- Heaps/Priority Queues
- Segment Trees
- Fenwick Trees
- Disjoint Set Union
