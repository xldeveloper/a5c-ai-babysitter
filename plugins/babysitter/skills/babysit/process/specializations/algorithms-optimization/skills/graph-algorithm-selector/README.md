# Graph Algorithm Selector

A skill for selecting optimal graph algorithms.

## Overview

This skill analyzes problem constraints and graph properties to recommend the most efficient algorithm for the given scenario.

## Key Features

- Constraint-based algorithm selection
- Complexity analysis
- Trade-off evaluation
- Special case handling
- Alternative suggestions

## Problem Types Supported

- Shortest Path
- Minimum Spanning Tree
- Connectivity
- Network Flow
- Matching
- Traversal
