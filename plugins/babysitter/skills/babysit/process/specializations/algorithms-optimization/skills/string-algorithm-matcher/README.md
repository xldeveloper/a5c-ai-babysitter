# String Algorithm Matcher

A skill for matching string problems to algorithms.

## Overview

This skill analyzes string processing problems and recommends the most appropriate algorithm based on the problem requirements and constraints.

## Key Features

- Pattern matching selection
- Suffix structure guidance
- Palindrome algorithm selection
- Rolling hash implementation
- Complexity analysis

## Algorithm Categories

### Pattern Matching
- KMP Algorithm
- Z-Algorithm
- Rabin-Karp
- Aho-Corasick

### Suffix Structures
- Suffix Array
- Suffix Tree
- Suffix Automaton

### Palindromes
- Manacher's Algorithm
- Hashing techniques
