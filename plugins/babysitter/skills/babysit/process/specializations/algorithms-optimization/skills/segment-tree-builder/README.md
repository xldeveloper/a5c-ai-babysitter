# Segment Tree Builder

A skill for generating customized segment tree implementations.

## Overview

This skill generates segment tree code for custom merge functions, supporting various variants including lazy propagation, persistence, and 2D extensions.

## Key Features

- Custom merge function support
- Lazy propagation generation
- Persistent variant support
- 2D segment tree generation
- Multiple language output

## Segment Tree Variants

- Basic (point update, range query)
- Lazy Propagation (range update)
- Persistent (version history)
- 2D Segment Tree
- Segment Tree Beats
