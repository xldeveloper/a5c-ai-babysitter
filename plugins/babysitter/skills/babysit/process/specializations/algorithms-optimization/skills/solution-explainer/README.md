# Solution Explainer

A skill for generating solution explanations.

## Overview

This skill generates clear, educational explanations of algorithm solutions, suitable for interview practice, learning, and documentation.

## Key Features

- Step-by-step walkthroughs
- Complexity analysis
- Alternative approaches
- Common mistake highlighting
- Interview-style formatting

## Explanation Components

1. Problem Understanding
2. Approach Overview
3. Algorithm Details
4. Complexity Analysis
5. Code Walkthrough
6. Edge Cases
7. Alternatives
