# Algorithm Visualizer

A skill for generating visual representations of algorithm execution.

## Overview

This skill creates step-by-step visualizations of algorithms to aid in understanding, debugging, and explaining algorithmic concepts to others.

## Key Features

- Step-by-step execution traces
- Multiple output formats (ASCII, SVG, GIF, HTML)
- Data structure state visualization
- DP table rendering
- Graph algorithm animation

## Supported Algorithms

- Sorting algorithms
- Search algorithms
- Graph traversals
- Dynamic programming
- Tree operations
