# Code Profiler

A skill for profiling algorithm performance.

## Overview

This skill profiles algorithm implementations to identify bottlenecks and suggest optimizations for time and memory usage.

## Key Features

- Runtime profiling
- Memory analysis
- Cache performance
- Hot spot detection
- Optimization suggestions

## Profiling Types

### Time Profiling
- Function timing
- Line profiling
- Call graphs

### Memory Profiling
- Allocation tracking
- Leak detection
- Peak usage
