# Interview Simulator

A skill for simulating coding interviews.

## Overview

This skill provides a realistic coding interview simulation with time constraints, progressive hints, and follow-up questions to help candidates prepare effectively.

## Key Features

- Time-boxed sessions
- Progressive hint system
- Follow-up question generation
- Communication evaluation
- Company-specific styles

## Interview Phases

1. Problem Presentation
2. Clarification
3. Approach Discussion
4. Implementation
5. Testing
6. Optimization
7. Follow-ups
