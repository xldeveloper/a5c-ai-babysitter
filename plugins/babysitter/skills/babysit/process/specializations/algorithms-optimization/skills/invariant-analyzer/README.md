# Invariant Analyzer

A skill for loop invariant analysis.

## Overview

This skill helps identify and verify loop invariants as part of constructing correctness proofs for algorithms.

## Key Features

- Loop invariant inference
- Invariant verification
- Pre/postcondition extraction
- Proof structure generation

## Invariant Properties

1. Initialization
2. Maintenance
3. Termination

## Common Patterns

- Range invariants
- Accumulator invariants
- Pointer invariants
- State invariants
