# GraphQL Mobile Skill

## Overview

The GraphQL Mobile skill provides GraphQL client integration capabilities for mobile applications, enabling type-safe API communication with caching and real-time subscriptions.

## Key Capabilities

- **Apollo Client**: React Native integration
- **Code Generation**: TypeScript/Swift/Kotlin types
- **Caching**: InMemoryCache, persistence
- **Subscriptions**: WebSocket real-time data
- **Optimistic UI**: Instant feedback patterns

## Quick Start

### React Native Setup

```bash
npm install @apollo/client graphql
npm install -D @graphql-codegen/cli
```

## Process Integration

| Process | Usage |
|---------|-------|
| `graphql-apollo-integration.js` | GraphQL implementation |
| `offline-first-architecture.js` | Offline caching |
