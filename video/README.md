# Babysitter Explainer Video

A 45-second Remotion video explaining Babysitter to Claude Code users.

## Quick Start

```bash
# Install dependencies
npm install

# Start development preview
npm run dev

# Render final video
npm run build
```

## Scenes

| Scene | Duration | Content |
|-------|----------|---------|
| Hook | 0:00-0:04 | "Your AI agent is 80% reliable" |
| Compounding Error | 0:04-0:14 | Quality degrades: 80%^5 = 33% |
| Quality Convergence | 0:14-0:24 | Babysitter loops until success |
| Resumability | 0:24-0:32 | Session state preserved |
| Breakpoints | 0:32-0:40 | Human approval before deploy |
| CTA | 0:40-0:45 | Logo + install command |

## Output

Rendered video will be saved to `out/babysitter-explainer.mp4`
